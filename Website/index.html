Your phone
is a server.

A minimal guide to transforming any Android device into a headless, secure, always-on server using open-source tools.

Termux OpenSSH Tailscale Wakelock Nginx Cloudflared

Full reproducible infrastructure guide is in docs/REPLICATION_GUIDE.md in this repository.

Begin Setup
Before you begin

Prerequisites

Android Device

Any Android phone or tablet running Android 7.0+. An old phone works perfectly.

Termux from F-Droid

Install Termux from F-Droid only. The Play Store version is deprecated and will not work.

Tailscale Account

Create a free account at tailscale.com. Install the official Tailscale Android app.

Step-by-step

Setup Guide

01

Update Termux

Open Termux and update all packages to their latest versions. This ensures you have the most recent tools and security patches.

bash 
pkg update && pkg upgrade -y
Press y if prompted to confirm any configuration changes during the upgrade.
02

Install OpenSSH

Install the OpenSSH package to enable secure remote shell access. Termux's SSH server runs on port 8022 by default (not the standard port 22).

bash 
# Install OpenSSH
pkg install openssh -y

# Set a password for SSH login
passwd

# Start the SSH daemon
sshd

# Check your username (you'll need this to connect)
whoami
Your Termux username is typically something like u0_a123. Note it down.
03

Acquire Wakelock

Android aggressively kills background processes. Acquire a wakelock to prevent the CPU from sleeping and keep your server available 24/7.

bash 
# Acquire wakelock (keeps CPU active)
termux-wake-lock

# To release it later:
# termux-wake-unlock
Also go to Settings → Apps → Termux → Battery and set it to "Unrestricted" to prevent Android from killing the app.
04

Connect via Tailscale

Install the official Tailscale Android app from the Play Store and enable it. This creates a secure mesh VPN — no port forwarding needed. Your phone gets a stable private IP accessible from any device on your Tailscale network.

bash 
# From your remote machine, connect:
ssh -p 8022 your_username@your-tailscale-ip

# Example:
# ssh -p 8022 u0_a123@100.64.0.2
Find your Tailscale IP in the Tailscale app or at login.tailscale.com/admin/machines.
05

Auto-start on Boot

Install Termux:Boot from F-Droid to automatically start your services when the device reboots. Open the Termux:Boot app once after installing to grant permissions.

bash 
# Create boot directory
mkdir -p ~/.termux/boot

# Create startup script
cat > ~/.termux/boot/start-server.sh << 'EOF'
#!/data/data/com.termux/files/usr/bin/sh
termux-wake-lock
sshd
EOF

# Make it executable
chmod 700 ~/.termux/boot/start-server.sh
Install both Termux and Termux:Boot from F-Droid only. Mixing sources (Play Store + F-Droid) will fail due to mismatched signing keys.
